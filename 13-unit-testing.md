# Unit Testing in C#

## xUnit
A popular testing framework for .NET.

### Example:
```csharp
[Fact]
public void TestMethod() {
    Assert.True(true);
}
```

## NUnit
Another widely used unit testing framework.

## Moq
A library to create mock objects for testing.

## Test Patterns
Learn about common test patterns such as AAA (Arrange, Act, Assert).

## Exercises
1. Write unit tests for a basic function.
2. Mock dependencies when testing services.
